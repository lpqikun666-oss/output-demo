## AI 遥感场景分类 Demo

本项目提供了一个基于 PyTorch + ResNet50 的 AI 场景分类演示，展示深度学习在地理空间分析中的应用。

详见 [scene_classifier.py](scene_classifier.py)。

### 技术栈

- Python / PyTorch / torchvision
- ResNet50 预训练模型（迁移学习）
- OpenCV 摄像头采集
- Matplotlib 可视化

### 用法

```bash
pip install -r requirements.txt
python scene_classifier.py
```
